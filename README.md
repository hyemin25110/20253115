# 20253115 김혜민 — Linux Command Table

**1.top**
현재 시스템의 CPU/메모리 사용량과 실행 중인 프로세스 상태를 _실시간으로_ 보여준다.
주요 기능:
•	시스템 전체 자원 사용률 모니터링
•	프로세스 목록 실시간 갱신
•	프로세스 우선순위 변경 가능
•	실시간 서버 상태 점검 활용


| **명령어** | **설명** | **옵션** |
|:---------:|:---------|:---------:|
| **top** | 실시간 시스템 모니터링 | - |
| **ps** | 프로세스 목록 출력 | `aux` |
| **jobs** | 백그라운드 작업 목록 | - |
| **kill** | 프로세스 종료 | `PID` |
