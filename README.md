# 20253115 김혜민 — Linux Command Table


## 1. top
__시스템의 실시간 프로세스 & 자원 사용량 모니터링 도구_
| **명령어** | **설명** | **대표 옵션** |
|:---------:|:---------|:--------------:|
| **top** | 실시간 시스템의 프로세스/CPU/메모리 사용량을 모니터링하는 명령어 | `-d`, `-p`, `-u` |


## 2. ps
__현재 실행 중인 프로세스 목록을 보여줌_
| **명령어** | **설명** | **대표 옵션** |
|:---------:|:---------|:--------------:|
| **ps** | 현재 실행 중인 프로세스를 출력하는 명령어 | `aux`, `-e`, `-f` |


## 3. jobs
__백그라운드 작업 목록 확인_
| **명령어** | **설명** | **대표 옵션** |
|:---------:|:---------|:--------------:|
| **jobs** | 백그라운드에서 실행 중인 작업 목록을 표시 | `-l`, `-n`, `-p` |


## 4. kill
__프로세스 종료 명령_
| **명령어** | **설명** | **대표 옵션 / 사용 예시** |
|:---------:|:---------|:--------------------------:|
| **kill** | 특정 프로세스(PID)에 시그널을 보내 종료하는 명령어 | `kill -9 PID`, `kill -15 PID` |


--------------------------------------------------
| **명령어** | **설명** | **옵션** |
|:---------:|:---------|:---------:|
| **top** | 실시간 시스템 모니터링 | - |
| **ps** | 프로세스 목록 출력 | `aux` |
| **jobs** | 백그라운드 작업 목록 | - |
| **kill** | 프로세스 종료 | `PID` |
